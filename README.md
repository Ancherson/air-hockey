# AIR HOCKEY

## Presentation

It's a video-game inspired by the real air-hockey !

## Usage

### Build

```
git clone https://gaufre.informatique.univ-paris-diderot.fr/rougeoll/air-hockey.git
cd air-hockey/Projet
./gradlew build
```
### Run

To run the server :
```
./gradlew runServer
```
To run the client :
```
./gradlew run
```
